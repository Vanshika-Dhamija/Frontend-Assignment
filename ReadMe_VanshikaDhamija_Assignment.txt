At first we have to install the node modules so for that we will write npm install in the terminal so that the node modules are installed.
Once node modules are insatlled we will write npm start.
The app will load on the localhost3000 in the browser.
On the left side paste the UI schema for any form you want to create.
Click the Submit button which is below the text area on the left hand side.
After clicking submit the dynamic form will be generated on the right hand side of the page.
Fill the form with your own choice and click the Submit button below the form after filling.
A page of the user profile will be displayed which shows all the entries which were filled by the user.
